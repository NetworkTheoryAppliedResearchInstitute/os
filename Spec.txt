## 1. Executive Summary

### 1.1 Purpose

NTARI OS is a Linux distribution based on Alpine Linux, designed to operate Small Office/Home Office (SOHO) nodes within an integrated mesh network. The system enables:

- **Peer-to-peer economic coordination** where hardware owners are prosumers
- **Community-owned infrastructure** replacing extractive platform models
- **Democratic governance** of network resources and policies
- **Automated capability discovery** and job matching
- **Cooperative storage** replacing centralized cloud services

### 1.2 Design Principles

1. **No Extraction**: Economic benefits accrue to participants, not platform owners
2. **Mutual Benefit**: Incentive structures assume good faith and reward cooperation
3. **Democratic**: Governance is transparent and community-controlled
4. **Dignity**: Exit is honored, participation is voluntary
5. **Hardware Longevity**: System optimizes for equipment lifespan, not planned obsolescence
6. **Privacy by Design**: Explicit consent required for sensitive hardware access

### 1.3 Target Hardware

**Reference Platform**: Intel i5-8500T (6 cores), 8GB RAM, x86_64
**Supported Architectures**: x86_64, ARM64, ARMv7 (32-bit)
**Minimum Requirements**: 1 CPU core, 512MB RAM, 1GB storage
**Typical Deployment**: Raspberry Pi 4, consumer desktops, refurbished hardware

### 1.4 Key Innovations

- **ROS2-inspired topology** for distributed systems coordination
- **Tiered storage strategy** optimizing for write patterns and media type
- **Positive exit incentives** using economic rewards, not penalties
- **Blockchain-based time consensus** via transaction "tics"
- **Automatic capability discovery** and network onboarding

## 2. System Architecture

### 2.1 Layer Overview

```
┌─────────────────────────────────────────────────────────────┐
│ Layer 5: Applications (Agrinet, Manufacturing, etc.)        │
├─────────────────────────────────────────────────────────────┤
│ Layer 4: Economic Coordination (SoHoLINK)                   │
├─────────────────────────────────────────────────────────────┤
│ Layer 3: Service Layer (Storage, DNS, Time Sync)            │
├─────────────────────────────────────────────────────────────┤
│ Layer 2: Communication (DDS Middleware, Topic Management)   │
├─────────────────────────────────────────────────────────────┤
│ Layer 1: Base OS (Alpine Linux + NTARI Extensions)          │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 Layer 1: Base Operating System

**Components:**
- Alpine Linux (musl libc, BusyBox)
- Kernel: Linux 6.x with mesh networking support (batman-adv, 802.11s)
- Package Manager: apk with NTARI repository
- Init System: OpenRC
- Security: Mandatory Access Control via AppArmor

**NTARI Extensions:**
```
/usr/bin/ntari              # CLI tool
/etc/ntari/                 # Configuration directory
/opt/ntari/                 # Application directory
/var/lib/ntari/             # State and data
/var/log/ntari/             # Logging
```

### 2.3 Layer 2: Communication Layer (DDS Middleware)

**DDS Implementation:**
- **Primary**: Cyclone DDS (Eclipse Foundation, LGPL)
- **Alternative**: Fast DDS (eProsima, Apache 2.0)
- **Abstraction**: NTARI RMW (ROS2 Middleware) adaptation layer

**DDS Domain Allocation:**
```yaml
domains:
  0: system           # Core OS functions
  1: agrinet          # Agricultural protocol
  2: manufacturing    # 3D printing networks
  3: logistics        # Delivery/drone coordination
  4: governance       # Democratic decision-making
  5-99: application   # Reserved for future apps
  100-232: user       # User-defined domains
```

**DDS Configuration Manager:**
```
/etc/ntari/dds-config.yaml
/usr/bin/ntari-dds-manager
```

### 2.4 Layer 3: Mandatory Service Layer

**Core Services:**

1. **Distributed Storage** (`ntari-storage-daemon`)
   - Federated file system
   - Erasure coding and replication
   - Tier management
   - Health monitoring

2. **Local DNS** (`ntari-dns-resolver`)
   - Mesh-local name resolution
   - Integration with traditional DNS
   - Caching layer

3. **Time Synchronization** (`ntari-time-sync`)
   - Transaction tic generation
   - Blockchain-based consensus
   - NTP fallback for internet-connected nodes

4. **Network Directory** (`ntari-directory-service`)
   - Node discovery
   - Capability aggregation
   - Geographic indexing

### 2.5 Layer 4: Economic Coordination (SoHoLINK)

**Components:**

1. **Resource Broker** (`soholink-broker`)
   - Matches jobs to capabilities
   - Manages bidding/offer system
   - Handles contracts

2. **Agent Profile Manager** (`soholink-agent`)
   - User skills and availability
   - Portfolio management
   - Reputation tracking

3. **Payment Processor** (`soholink-payments`)
   - Transaction logging
   - Multi-currency support
   - Escrow for contracts

4. **Reputation Engine** (`soholink-reputation`)
   - LBTAS score calculation
   - Dispute resolution tracking

### 2.6 Layer 5: Application Framework

**Application Structure:**
```
/opt/ntari/apps/{app_name}/
├── manifest.yaml       # DDS requirements, resource needs
├── schema/            # Message type definitions
├── bin/               # Executables
├── config/            # Default configuration
└── data/              # Application data
```

**Application Manifest Example:**
```yaml
name: agrinet
version: 1.0.0
dds_requirements:
  implementation: cyclone
  domain: 1
  qos_profile: reliable
resource_requirements:
  compute: 0.5  # cores
  memory: 256   # MB
  storage: 100  # MB
  network: 1    # Mbps
permissions:
  - /network/storage/read
  - /soho/{uuid}/resources/sensors
```

### 4.1 Tiered Storage Strategy

#### 4.1.1 Storage Tier Definitions

**Tier 1: Active/Volatile → HDD**

*Purpose*: Frequently modified files  
*Media*: HDD (hard disk drives)  
*Redundancy*: Replication (3x)  
*Write Pattern*: > 10 writes/day

#### 4.1.3 Erasure Coding Schemes

**Reed-Solomon Implementation:**

```python
# Erasure coding parameters
SCHEMES = {
    "light": {
        "data_shards": 6,
        "parity_shards": 2,
        "overhead": 1.33,
        "fault_tolerance": 2
    },
    "medium": {
        "data_shards": 8,
        "parity_shards": 4,
        "overhead": 1.5,
        "fault_tolerance": 4
    },
    "heavy": {
        "data_shards": 20,
        "parity_shards": 10,
        "overhead": 1.5,
        "fault_tolerance": 10
    }
}
```

**Shard Distribution Algorithm:**

```python
def distribute_shards(file_shards, node_list):
    """
    Distribute shards ensuring:
    1. No two shards of same file on same node
    2. Geographic diversity for data shards
    3. Proximity for parity shards (faster reconstruction)
    """
    data_shards = file_shards[:data_shard_count]
    parity_shards = file_shards[data_shard_count:]
    
    # Sort nodes by distance from file owner
    nodes_sorted = sort_by_distance(node_list, owner_location)
    
    # Place data shards geographically distributed
    data_nodes = select_diverse_nodes(nodes_sorted, len(data_shards))
    
    # Place parity shards on nearby high-compute nodes
    parity_nodes = select_compute_nodes(nodes_sorted, len(parity_shards))
    
    return zip(file_shards, data_nodes + parity_nodes)
```

**Reconstruction Process:**

```python
def reconstruct_shard(file_id, missing_shard_index):
    """
    Reconstruct a missing shard using Reed-Solomon math
    """
    # 1. Identify file's shard configuration
    config = get_file_config(file_id)
    scheme = SCHEMES[config.erasure_scheme]
    
    # 2. Gather available shards
    available = fetch_available_shards(file_id)
    
    # 3. Need at least data_shard_count to reconstruct
    if len(available) < scheme["data_shards"]:
        raise InsufficientShards(
            f"Need {scheme['data_shards']}, have {len(available)}"
        )
    
    # 4. Perform Reed-Solomon reconstruction
    reconstructed = reed_solomon_decode(available, missing_shard_index)
    
    # 5. Verify hash
    if hash(reconstructed) != get_shard_hash(file_id, missing_shard_index):
        raise IntegrityError("Reconstructed shard hash mismatch")
    
    # 6. Store reconstructed shard
    new_node = select_storage_node()
    store_shard(new_node, file_id, missing_shard_index, reconstructed)
    
    return new_node
```

```python
class HealthMonitor:
    def __init__(self):
        self.check_interval = 3600  # 1 hour
        self.drives = self.detect_drives()
        
    def monitor_loop(self):
        while True:
            for drive in self.drives:
                health = self.check_smart(drive)
                
                if health.status == "warning":
                    self.reduce_tier_allocation(drive)
                    self.notify_user(drive, health)
                    
                elif health.status == "failing":
                    self.migrate_all_data(drive)
                    self.disable_drive(drive)
                    self.alert_user_urgent(drive, health)
                    
                self.publish_health_status(drive, health)
                
            time.sleep(self.check_interval)
    
    def check_smart(self, drive):
        """Read SMART data and assess health"""
        smart = subprocess.run(
            ["smartctl", "-A", drive.device],
            capture_output=True
        )
        
        return {
            "device": drive.device,
            "status": self.assess_status(smart.stdout),
            "power_on_hours": self.extract_value(smart, "Power_On_Hours"),
            "reallocated_sectors": self.extract_value(smart, "Reallocated_Sector_Ct"),
            "wear_level": self.extract_value(smart, "Wear_Leveling_Count") if drive.type == "SSD" else None,
            "temperature": self.extract_value(smart, "Temperature_Celsius")
        }
```

**Tier Rebalancing on Drive Degradation:**

```python
def reduce_tier_allocation(self, drive):
    """
    When drive shows warning signs:
    1. Stop accepting new writes
    2. Reduce tier assignments
    3. Prefer read-only tiers
    """
    if drive.type == "SSD" and drive.wear_level > 80:
        # Move from write-heavy to read-only tiers
        self.migrate_tier(drive, from_tier=4, to_tier=2)
        self.migrate_tier(drive, from_tier=1, to_tier=2)
        
    elif drive.type == "HDD" and drive.reallocated_sectors > 10:
        # Reduce load on failing HDD
        self.reduce_allocation(drive, by_percent=50)
```

### 5.2 Resource Broker

**Job Matching Algorithm:**

```python
class ResourceBroker:
    def match_job_to_nodes(self, job):
        """
        Match job requirements to available nodes
        Returns ranked list of suitable nodes
        """
        # 1. Filter by hard requirements
        candidates = self.filter_by_requirements(job)
        
        # 2. Score candidates
        scored = []
        for node in candidates:
            score = self.calculate_match_score(job, node)
            scored.append((node, score))
        
        # 3. Sort by score descending
        scored.sort(key=lambda x: x[1], reverse=True)
        
        # 4. Return top N candidates
        return scored[:10]
    
    def calculate_match_score(self, job, node):
        """
        Scoring factors:
        - Capability match (0-40 points)
        - Reputation (0-25 points)
        - Price competitiveness (0-20 points)
        - Proximity (0-10 points)
        - Availability (0-5 points)
        """
        score = 0
        
        # Capability match
        if job.requirements.compute:
            if node.resources.compute.cores >= job.requirements.compute.cores:
                score += 20
            if node.resources.compute.ram_gb >= job.requirements.compute.ram_gb:
                score += 20
        
        # Reputation
        score += node.reputation.overall * 5  # 0-5 rating * 5 = 0-25 points
        
        # Price
        if job.budget:
            node_bid = node.pricing.estimate_bid(job)
            if node_bid <= job.budget.max:
                price_ratio = node_bid / job.budget.max
                score += (1 - price_ratio) * 20  # Lower price = higher score
        
        # Proximity (if location matters)
        if job.requirements.location:
            distance_km = calculate_distance(
                job.requirements.location.coordinates,
                node.profile.location.coordinates
            )
            if distance_km <= job.requirements.location.radius_km:
                score += 10 * (1 - distance_km / job.requirements.location.radius_km)
        
        # Availability
        if node.schedule.available_now:
            score += 5
        
        return score
```

```python
class BidProcessor:
    def submit_bid(self, soho_id, job_id, amount, details):
        """Submit a bid to a job"""
        bid = {
            "bid_id": generate_uuid(),
            "job_id": job_id,
            "from_soho": soho_id,
            "amount": amount,

            "submitted": int(time.time()),

            "status": "pending"

        # Publish to DDS
        self.dds.publish(f"/soho/{soho_id}/bids/outgoing", bid)
        
        # Notify job poster
        job = self.get_job(job_id)
        self.notify_node(job.posted_by, "new_bid", bid)
        
        return bid
    
    def accept_bid(self, job_id, bid_id):
        """Job poster accepts a bid"""
        bid = self.get_bid(bid_id)
        job = self.get_job(job_id)
        
        # Create contract
        contract = self.create_contract(job, bid)
        
        # If escrow required, lock funds
        if job.escrow_required:
            self.escrow.lock_funds(
                from_soho=job.posted_by,
                amount=bid.amount,
                contract_id=contract.contract_id
            )
        
        # Update bid status
        bid.status = "accepted"
        self.dds.publish(f"/soho/{bid.from_soho}/bids/outgoing", bid)
        
        # Update job status
        job.status = "in_progress"
        job.assigned_to = bid.from_soho
        self.dds.publish("/network/market/requests", job)
        
        # Notify both parties
        self.notify_node(bid.from_soho, "bid_accepted", contract)
        self.notify_node(job.posted_by, "contract_created", contract)
        
        return contract
```

```python
class ScheduleManager:

        """
        Check if node can accommodate job within deadline

        agent = self.get_agent_profile(soho_id)
        schedule = agent.schedule

            "hours_reserved": hours,
            "reserved_at": int(time.time())
        })
```

**Escrow Service:**

```python
class EscrowService:
    def lock_funds(self, from_soho, amount, contract_id):
        """Lock funds in escrow"""
        escrow = {
            "escrow_id": generate_uuid(),
            "contract_id": contract_id,
            "from_soho": from_soho,
            "amount": amount,
            "locked_at": int(time.time()),
            "status": "locked",
            "release_conditions": {
                "contract_completion": True,
                "mutual_agreement": True,
                "dispute_resolution": True
            }
        }
        
        # Deduct from sender's balance
        self.deduct_balance(from_soho, amount)
        
        # Store in escrow account
        self.escrow_store[escrow.escrow_id] = escrow
        
        return escrow
    
    def release(self, contract_id, to_soho, amount, currency):
        """Release escrowed funds"""
        escrow = self.get_escrow_by_contract(contract_id)
        
        if escrow.status != "locked":
            raise EscrowError("Funds not locked")
        
        # Transfer to recipient
        self.add_balance(to_soho, amount, currency)
        
        # Mark escrow released
```

### 5.5 Reputation Engine (LBTAS)

**LBTAS Score Calculation:**

```python
class ReputationEngine:
    """
    Leveson-Based Trade Assessment Scale
    https://github.com/NTARI-RAND/Leveson-Based-Trade-Assessment-Scale
    """
    
    def calculate_lbtas(self, soho_id):
        """Calculate overall LBTAS score"""
        completed = self.get_completed_contracts(soho_id)
        
        if not completed:
            return None  # No history yet
        
        scores = {
            "reliability": self.calc_reliability(completed),
            "quality": self.calc_quality(completed),
            "communication": self.calc_communication(completed),
            "safety": self.calc_safety(soho_id)
        }
        
        # Weighted average
        overall = (
            scores["reliability"] * 0.30 +
            scores["quality"] * 0.30 +
            scores["communication"] * 0.20 +
            scores["safety"] * 0.20
        )
        
        return {
            "overall": round(overall, 2),
            "categories": scores,
            "sample_size": len(completed),
            "methodology": "https://github.com/NTARI-RAND/Leveson-Based-Trade-Assessment-Scale"
        }
    
    def calc_reliability(self, contracts):
        """
        Reliability: On-time completion, uptime
        """
        on_time = sum(1 for c in contracts if c.completed_on_time)
        total = len(contracts)
        
        ratio = on_time / total if total > 0 else 0
        
        # Convert to 0-5 scale
        return ratio * 5
    
    def calc_quality(self, contracts):
        """
        Quality: Client ratings, revision requests
        """
        ratings = [c.client_rating for c in contracts if c.client_rating]
        
        if not ratings:
            return 3.0  # Neutral if no ratings
        
        return sum(ratings) / len(ratings)
    
    def calc_communication(self, contracts):
        """
        Communication: Response time, clarity
        """
        response_times = [c.avg_response_time_hours for c in contracts]
        
        # Scoring: < 1hr = 5, 1-4hr = 4, 4-24hr = 3, 24-48hr = 2, >48hr = 1
        scores = []
        for rt in response_times:
            if rt < 1:
                scores.append(5)
            elif rt < 4:
                scores.append(4)
            elif rt < 24:
                scores.append(3)
            elif rt < 48:
                scores.append(2)
            else:
                scores.append(1)
        
        return sum(scores) / len(scores) if scores else 3.0
    
    def calc_safety(self, soho_id):
        """
        Safety: Security incidents, data breaches
        """
        incidents = self.get_security_incidents(soho_id)
        
        if not incidents:
            return 5.0  # Perfect if no incidents
        
        # Deduct points for each incident type
        score = 5.0
        for incident in incidents:
            if incident.severity == "critical":
                score -= 2.0
            elif incident.severity == "major":
                score -= 1.0
            elif incident.severity == "minor":
                score -= 0.5
        
        return max(0, score)
```

## 6. Exit & Legacy Handoff Protocol

### 6.1 Exit Types

**Graceful Exit:**
- Node announces exit with advance notice (1-30 days)
- Data handed off to recipient nodes
- Eligible for exit bonuses

**Position Transfer:**
- Node sells position to another node
- All resources, reputation, contracts transferred
- Market-determined price

**Immediate Exit:**
- Node stops without handoff
- Network triggers emergency replication
- No bonuses, no penalties

### 6.2 Graceful Exit Flow

**State Machine:**

```
1. OPERATING
   ↓ (user initiates exit)
2. EXIT_ANNOUNCED
   ↓ (announcement bonus paid)
3. HANDOFF_IN_PROGRESS
   ↓ (shards transferring)
4. VERIFICATION
   ↓ (recipients verify shards)
5. HANDOFF_COMPLETE
   ↓ (completion bonus paid)
6. EXITED
```

**Implementation:**

```python
class ExitManager:
    def initiate_graceful_exit(self, soho_id, handoff_period_hours):
        """
        User initiates graceful exit
        """
        node = self.get_node(soho_id)
        
        # Calculate announcement bonus
        max_bonus = 2.50
        bonus_multiplier = min(handoff_period_hours / 168, 1.0)  # 7 days = 100%
        announcement_bonus = max_bonus * bonus_multiplier
        
        # Create exit record
        exit_record = {
            "soho_id": soho_id,
            "exit_type": "graceful",
            "announced_at": int(time.time()),
            "handoff_period_hours": handoff_period_hours,
            "expected_completion": int(time.time()) + handoff_period_hours * 3600,
            "announcement_bonus": announcement_bonus,
            "status": "announced",
            "affected_resources": {
                "storage_shards": len(node.storage.stored_shards),
                "active_contracts": len(node.contracts.active),
                "governance_delegations": len(node.governance.delegations)
            }
        }
        
        # Publish exit intent
        self.dds.publish(
            f"/soho/{soho_id}/network/exit_intent",
            exit_record
        )
        
        # Pay announcement bonus immediately
        self.payment.process_exit_bonus(
            soho_id=soho_id,
            amount=announcement_bonus,
            bonus_type="announcement"
        )
        
        # Begin handoff orchestration
        self.orchestrate_handoff(exit_record)
        
        return exit_record
    
    def orchestrate_handoff(self, exit_record):
        """
        Coordinate shard transfer to recipient nodes
        """
        soho_id = exit_record.soho_id
        shards = self.get_node_shards(soho_id)
        
        # Select recipient nodes
        recipients = self.select_recipients(
            shard_count=len(shards),
            exclude=[soho_id]
        )
        
        # Create transfer plan
        transfer_plan = []
        for shard, recipient in zip(shards, recipients):
            transfer_plan.append({
                "shard_id": shard.shard_id,
                "file_id": shard.file_id,
                "from_soho": soho_id,
                "to_soho": recipient.soho_id,
                "size_bytes": shard.size_bytes,
                "priority": self.calculate_priority(shard)
            })
        
        # Sort by priority (critical files first)
        transfer_plan.sort(key=lambda x: x["priority"], reverse=True)
        
        # Execute transfers
        for transfer in transfer_plan:
            self.execute_shard_transfer(transfer)
            self.update_handoff_progress(soho_id, transfer)
        
        # Monitor until complete
        self.monitor_handoff(soho_id)
    
    def execute_shard_transfer(self, transfer):
        """Transfer individual shard to recipient"""
        # Fetch shard from exiting node
        shard_data = self.fetch_shard(
            soho_id=transfer.from_soho,
            shard_id=transfer.shard_id
        )
        
        # Verify integrity
        if hash(shard_data) != transfer.expected_hash:
            raise IntegrityError(f"Shard {transfer.shard_id} hash mismatch")
        
        # Send to recipient
        self.send_shard(
            soho_id=transfer.to_soho,
            shard_id=transfer.shard_id,
            data=shard_data
        )
        
        # Recipient verifies
        verification = self.request_verification(
            soho_id=transfer.to_soho,
            shard_id=transfer.shard_id
        )
        
        if not verification.success:
            raise TransferError(f"Recipient failed to verify shard {transfer.shard_id}")
        
        # Update shard location map
        self.update_shard_location(
            shard_id=transfer.shard_id,
            old_location=transfer.from_soho,
            new_location=transfer.to_soho
        )
    
    def complete_handoff(self, soho_id):
        """
        Finalize exit and pay completion bonuses
        """
        exit_record = self.get_exit_record(soho_id)
        
        # Calculate bonuses
        completion_bonus = 5.00
        
        # Time bonus if completed early
        time_saved = exit_record.expected_completion - int(time.time())
        if time_saved > 0:
            time_bonus_rate = 0.10  # $0.10 per hour saved
            time_bonus = min(time_saved / 3600 * time_bonus_rate, 2.00)
        else:
            time_bonus = 0
        
        # Verification bonus (all shards verified)
        verification_bonus = 1.50 if exit_record.all_shards_verified else 0
        
        total_bonus = completion_bonus + time_bonus + verification_bonus
        
        # Pay bonuses
        self.payment.process_exit_bonus(
            soho_id=soho_id,
            amount=total_bonus,
            bonus_type="completion",
            breakdown={
                "completion": completion_bonus,
                "time": time_bonus,
                "verification": verification_bonus
            }
        )
        
        # Award badges
        self.reputation.award_badge(soho_id, "reliable_steward")
        
        # Update reputation
        rep = self.reputation.get_reputation(soho_id)
        rep.graceful_exits += 1
        if exit_record.all_shards_verified:
            rep.exit_reliability_score = 5.0
        self.reputation.update_reputation(soho_id, rep)
        
        # Mark exit complete
        exit_record.status = "completed"
        exit_record.completed_at = int(time.time())
        exit_record.total_bonuses = (
            exit_record.announcement_bonus + total_bonus
        )
        
        # Publish completion
        self.dds.publish(
            f"/soho/{soho_id}/network/exit_complete",
            exit_record
        )
        
        return exit_record
```

```python
            "seller": position.seller,

            "amount": bid.offer,
            "currency": position.currency,
            "position_details": position.position_details,
            "created": int(time.time()),
            "status": "pending_payment"
        }
        
        # Escrow payment
        self.escrow.lock_funds(
            from_soho=bid.buyer,
            amount=bid.offer,
            contract_id=contract.contract_id
        )
        
        # Initiate transfer
        self.execute_position_transfer(contract)
        
        return contract
    
    def execute_position_transfer(self, contract):
        """Transfer position from seller to buyer"""
        seller = contract.seller
        buyer = contract.buyer
        
        # 1. Transfer storage shards
        self.transfer_all_shards(from_soho=seller, to_soho=buyer)
        
        # 2. Transfer active contracts
        self.reassign_contracts(from_soho=seller, to_soho=buyer)
        
        # 3. Transfer partial reputation
        seller_rep = self.reputation.get_reputation(seller)
        buyer_rep = self.reputation.get_reputation(buyer)
        
        # Buyer inherits 50% of seller's reputation
        buyer_rep.overall = (buyer_rep.overall + seller_rep.overall * 0.5) / 1.5
        buyer_rep.badges.extend(
            [b for b in seller_rep.badges if b.transferable]
        )
        
        self.reputation.update_reputation(buyer, buyer_rep)
        
        # 4. Release payment to seller
        self.escrow.release(
            contract_id=contract.contract_id,
            to_soho=seller,
            amount=contract.amount,
            currency=contract.currency
        )
        
        # 5. Update position transfer status
        contract.status = "completed"
        contract.completed_at = int(time.time())
        
        # 6. Notify both parties
        self.notify_node(seller, "position_sold", contract)
        self.notify_node(buyer, "position_acquired", contract)
        
        return contract
```

```python
class EmergencyExitHandler:
    HEARTBEAT_TIMEOUT = 180  # 3 minutes
    
    def monitor_heartbeats(self):
        """Continuously monitor for missing heartbeats"""
        while True:
            now = int(time.time())
            
            for node in self.get_all_nodes():
                last_heartbeat = node.last_heartbeat_time
                
                if now - last_heartbeat > self.HEARTBEAT_TIMEOUT:
                    self.handle_presumed_failure(node.soho_id)
            
            time.sleep(60)
    
    def handle_presumed_failure(self, soho_id):
        """Handle node that stopped responding"""
        node = self.get_node(soho_id)
        
        # Mark as presumed failed
        node.status = "presumed_failed"
        node.failed_at = int(time.time())
        
        # Trigger emergency replication
        self.trigger_emergency_replication(soho_id)
        
        # Don't penalize reputation (could be hardware failure)
        # Just note it happened
        rep = self.reputation.get_reputation(soho_id)
        rep.unplanned_exits += 1
        self.reputation.update_reputation(soho_id, rep)
        
        # Notify network
        self.dds.publish(
            "/network/alerts/node_failure",
            {
                "soho_id": soho_id,
                "failed_at": node.failed_at,
                "affected_shards": len(node.storage.stored_shards),
                "recovery_initiated": True
            }
        )
    
    def trigger_emergency_replication(self, failed_soho):
        """Immediately begin replicating shards from failed node"""
        shards = self.get_node_shards(failed_soho)
        
        for shard in shards:
            # Check current redundancy
            locations = self.get_shard_locations(shard.shard_id)
            remaining = [loc for loc in locations if loc != failed_soho]
            
            file_config = self.get_file_config(shard.file_id)
            
            if file_config.redundancy_method == "replication":
                # Need to restore to target replication factor
                target = file_config.replication_factor
                current = len(remaining)
                
                if current < target:
                    # Urgent: below target replication
                    priority = "critical"
                    needed = target - current
                elif current == target:
                    # At target, but one node failed
                    priority = "high"
                    needed = 1
                else:
                    # Above target, no rush
                    priority = "normal"
                    needed = 0
            
            else:  # erasure coding
                scheme = SCHEMES[file_config.erasure_scheme]
                current = len(remaining)
                
                if current < scheme.data_shards:
                    # CRITICAL: Below reconstruction threshold
                    priority = "emergency"
                    needed = scheme.data_shards + scheme.parity_shards - current
                elif current < scheme.data_shards + scheme.parity_shards:
                    # High: Can reconstruct but need more redundancy
                    priority = "high"
                    needed = scheme.data_shards + scheme.parity_shards - current
                else:
                    # At target
                    priority = "normal"
                    needed = 1
            
            # Queue replication job
            if needed > 0:
                self.queue_replication(
                    shard_id=shard.shard_id,
                    source_nodes=remaining,
                    copies_needed=needed,
                    priority=priority
                )
```

```python
class MessageSigner:
    def sign_message(self, message_dict):
        """Sign DDS message with node's private key"""
        # Serialize message
        canonical = json.dumps(message_dict, sort_keys=True)
        
        # Sign with Ed25519
        signature = self.private_key.sign(canonical.encode())
        
        # Attach signature
        message_dict["signature"] = base64.b64encode(signature).decode()
        
        return message_dict
    
    def verify_message(self, message_dict, sender_public_key):
        """Verify message signature"""
        # Extract signature
        signature_b64 = message_dict.pop("signature")
        signature = base64.b64decode(signature_b64)
        
        # Canonical form
        canonical = json.dumps(message_dict, sort_keys=True)
        
        # Verify
        try:
            sender_public_key.verify(signature, canonical.encode())
            return True
        except:
            return False
```

```python
    def calc_quorum(self, scope):
        """Calculate required quorum based on scope"""
        if scope == "global":
            # 20% of all nodes
            return int(self.get_node_count() * 0.20)
        elif scope == "municipal":
            # 30% of municipal nodes
            return int(self.get_municipal_node_count() * 0.30)
        elif scope == "wan":
            # 25% of WAN nodes
            return int(self.get_wan_node_count() * 0.25)
        else:
            # Local: 50% of local mesh
            return int(self.get_local_node_count() * 0.50)
```

**Voting:**

```python
def cast_vote(self, soho_id, proposal_id, vote):
    """Cast a vote on proposal"""
    proposal = self.get_proposal(proposal_id)
    
    # Check voting period
    now = int(time.time())
    if now < proposal.voting_starts or now > proposal.voting_ends:
        raise VotingClosed("Voting period not active")
    
    # Check for duplicate vote
    existing = self.get_vote(soho_id, proposal_id)
    if existing:
        raise DuplicateVote("Already voted on this proposal")
    
    # Get voter's weight
    rep = self.reputation.get_reputation(soho_id)
    weight = rep.network_citizenship_tier_multiplier  # 1.0-1.5 based on tier
    
    # Record vote
    vote_record = {
        "proposal_id": proposal_id,
        "voter": soho_id,
        "vote": vote,  # "yes", "no", "abstain"
        "weight": weight,
        "cast_at": now,
        "signature": self.sign_vote(soho_id, proposal_id, vote)
    }
    
    # Update proposal tallies
    proposal.votes[vote] += 1
    proposal.weighted_votes[vote] += weight
    
    # Publish vote
    self.dds.publish(f"/network/governance/votes", vote_record)
    
    # Update proposal
    self.dds.publish("/network/governance/proposals", proposal)
    
    return vote_record
```

**Proposal Resolution:**

```python
def resolve_proposal(self, proposal_id):
    """Resolve proposal after voting ends"""
    proposal = self.get_proposal(proposal_id)
    
    # Check voting ended
    if int(time.time()) < proposal.voting_ends:
        raise VotingActive("Voting still in progress")
    
    # Check quorum
    total_votes = sum(proposal.votes.values())
    if total_votes < proposal.quorum_required:
        proposal.status = "rejected"
        proposal.rejection_reason = "Quorum not met"
        self.dds.publish("/network/governance/proposals", proposal)
        return proposal
    
    # Determine result (weighted votes)
    yes_votes = proposal.weighted_votes["yes"]
    no_votes = proposal.weighted_votes["no"]
    
    # Simple majority (can be configured per proposal type)
    if yes_votes > no_votes:
        proposal.status = "passed"
        proposal.implementation_date = int(time.time()) + 86400 * 7  # 7 days
        
        # Schedule implementation
        self.schedule_implementation(proposal)
    else:
        proposal.status = "rejected"
        proposal.rejection_reason = "Majority voted no"
    
    # Publish final result
    self.dds.publish("/network/governance/proposals", proposal)
    
    return proposal
```

### 9.2 Network Discovery

**Discovery Protocol:**

```python
class NetworkDiscovery:
    def discover_nodes(self):
        """Discover nearby NTARI nodes"""
        # 1. Listen for heartbeats on local mesh
        mesh_nodes = self.listen_mesh_heartbeats(timeout=30)
        
        # 2. Query WAN directory if internet available
        wan_nodes = self.query_wan_directory() if self.has_internet() else []
        
        # 3. Combine and deduplicate
        all_nodes = self.deduplicate(mesh_nodes + wan_nodes)
        
        # 4. Geo-sort by proximity
        sorted_nodes = self.sort_by_distance(all_nodes, self.get_location())
        
        return sorted_nodes
    
    def listen_mesh_heartbeats(self, timeout):
        """Listen for DDS heartbeats on mesh network"""
        discovered = []
        
        # Subscribe to heartbeat topic
        subscriber = self.dds.subscribe(
            topic="/soho/+/heartbeat",  # + is wildcard
            callback=lambda msg: discovered.append(msg)
        )
        
        # Wait for timeout
        time.sleep(timeout)
        
        subscriber.unsubscribe()
        
        return discovered
```

## 12. Appendices

### 12.1 Glossary

**SOHO**: Small Office/Home Office - a node in the NTARI network

**DDS**: Data Distribution Service - publish/subscribe middleware for real-time systems

**Tic**: Time consensus unit in transaction blockchain

**LBTAS**: Leveson-Based Trade Assessment Scale - reputation scoring system

**Shard**: Piece of a file in distributed storage (either data or parity)

**Erasure Coding**: Data redundancy technique using parity mathematics (like RAID)

**Replication**: Data redundancy by storing complete copies

**Tier**: Storage classification based on access pattern and media type

**Legacy Handoff**: Graceful exit process with data transfer

**Position Transfer**: Selling node position to another participant

**SoHoLINK**: Economic coordination layer enabling peer-to-peer job matching

**Mesh Network**: Decentralized network where nodes relay data for each other

**WAN**: Wide Area Network - interconnected SOHOs across larger geographic areas

**Municipal Network**: Local/city-level SOHO cluster

### 12.4 Troubleshooting Guide

**Common Issues:**

1. **Node not discovered by network**
   - Check DDS domain configuration
   - Verify firewall allows DDS ports (7400, 7401)
   - Confirm network connectivity

2. **Storage tier migration not working**
   - Check `/var/log/ntari/storage-daemon.log`
   - Verify `auto_tier = true` in config
   - Ensure sufficient CPU for erasure coding

3. **Hardware access denied**
   - Review `/var/log/ntari/hardware-access.log`
   - Check device permissions in `/etc/ntari/hardware-security.yaml`
   - Verify user is in `ntari-hardware` group

4. **Exit handoff stuck**
   - Check network connectivity to recipient nodes
   - Review shard transfer logs
   - Manually trigger completion: `ntari exit complete --force`
